## Dependencias backend

- Spring Data JPA
- MySQL Driver
- Spring Security
- Lombok
- Validation

## Levantar backend
1. Clonar el repositorio con `git clone https://github.com/ArielNicolas2021/Grupo6-DDS.git`
2. Navegar al directorio del backend: `cd backend`
3. Crear una base de datos MySQL llamada `gestion_gastos`
3. Tipear el comando `./mvnw spring-boot:run` para levantar el backend
